AI · Healthcare · IoT 01

DeepCalm — AI-Based Stress Monitoring & Therapy System

Real-time physiological stress detection with personalized AI therapy via ESP32 + Firebase

GitHub

DeepCalm is an AI-driven stress monitoring system that processes real-time physiological data using a Flask-based backend and ML model to predict stress levels. Sensor data collected via ESP32 is streamed to Firebase for cloud-based handling and real-time analysis.

The system dynamically delivers personalized therapy including breathing guidance and audio-based relaxation through an interactive web interface, with user-specific data storage and stress trend visualization.

ML · Cybersecurity · NLP 02

CogniCheck — Online Fraud Detection System

Detecting fake websites, fake news & fraudulent phone numbers with ML & NLP

GitHub

CogniCheck is a web-based system enhancing online safety by detecting fake websites, identifying fake news, and validating mobile numbers. Uses GaussianNB for website detection and LinearSVC with TF-IDF for news classification.

Integrates the NumLookup API to verify phone numbers and provide details like country and carrier. Fast, accurate results through a clean UI to help users avoid phishing, misinformation, and fraudulent calls.

GaussianNBLinearSVCTF-IDFNumLookup APIPythonFlaskNLP

Web App · Security · Civic Tech 03

VoteSecure — Online Voting System for College Students

Secure, transparent digital elections with RBAC, SQL injection prevention & real-time tallying

GitHub

The Online Voting System enables secure and efficient digital elections within an academic environment. Uses authentication mechanisms, session management, and password hashing for secure user access, with role-based access control for admins and students.

Built with PHP and MySQL, incorporates prepared statements to prevent SQL injection, enforces one-user-one-vote policy, and maintains real-time vote tallying for full transparency.
